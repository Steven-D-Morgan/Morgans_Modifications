# CUSTOM COMPONENTS
### [HACS](https://hacs.xyz/)
- If you want to install a modified version of Home Assistant that will include a ton of additional components.
- I havent used HACS in years, but I didnt personally like it. Im sure it has came a long way since then
### [pfSense](https://github.com/travisghansen/hass-pfsense)
- Get network entities and some useful services to your pfSense Router.
- Switches: It comes with 40 to do things like toggle your DHCP Server or DNS Resolver.
### [Netgear WAX](https://github.com/rroller/netgear)
- Sensors from your Netgear access points.
### [Alexa Media Player](https://github.com/alandtse/alexa_media_player)
- This is the HOLY GRAIL of custom components.
- Gives you damn near total usage of your Echo devices via Alexa's API.
### [Emporia Vue](https://github.com/magico13/ha-emporia-vue)
- Use your Emporia Sensors OEM API for energy monitoring.
